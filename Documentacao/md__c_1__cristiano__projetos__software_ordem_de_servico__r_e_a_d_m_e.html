Sobre

Projeto desenvolvido para informatizar o sistema de ordem de serviço, principalmente para a área de equipamentos eletrônicos.

Desenvolvido na plataforma .NET(Windows Forms), utilizando o .NET Framework 4.5.

Funcionalidades

  • Sistema de impressão de ordem de serviço via PDF.
  • Cadastro de Pessoa Física / Juridica.
  • Gerenciamento de ordem de serviço.
  • Gerenciamento de serviços básicos.
  • Gerenciamento de serviços finalizados.
  • Sistema de envio de E-mail informando término do serviço.
  • Sistema de técnicos.
  • Sistema de nível de acesso.
